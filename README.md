# Book Management

Aplikasi ini digunakan untuk mengelola data buku dan data pengarang.

## Kebutuhan

1. Laravel 10
2. Database MySQL
3. JavaScript
4. Bootstrap

## Cara Installasi

1. **Clone Repository**

   ```bash
   git clone <URL_REPOSITORY>
   cd <NAMA_FOLDER_REPOSITORY>

2. **Import database "book" pada folder db**
3. **Jalankan dengan perintah**
   ```bash
   php artisan serve
